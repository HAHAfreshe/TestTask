# TestTask
Программа автозамены текста во всех файлах каталога. 

Программе в качестве аргументов передаются:
1) путь к каталогу с файлами
2) текст, который нужно заменить 
3) текст для замены.

Для каждого запуска программы создавается свой log-файл, содержащий историю автозамен: название файла, позиции файле, где были произведены замены, и примерный текст «как было» -> «как стало».
